# KeepSake
<p style="text-align: center;">Keepsake is a cloud-based artefacts registry that allows users to connect to their family network, catelogue and share of all their priceless family heirlooms. This online registry allows for users to preserve their familyâ€™s history with a focus on ease of use and privacy of information. It was created using agile development methodologies and supports primarily Android.<br><br>
This repository contains the code for Keepsake's front end and back end functionality. </p>
